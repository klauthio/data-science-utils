# 🛠 Data Science Toolbox

Repositorio de utilidades personalizadas para agilizar procesos de **limpieza, análisis y visualización de datos**. 
Este proyecto nace como parte de mi formación en la carrera de **Técnico en Ciencia de Datos**.

## 📂 Estructura del Proyecto

* **`cleaning/`**: Funciones para tratamiento de valores nulos, estandarización de nombres (ETL) y manejo de tipos de datos con Pandas.
* **`analysis/`**: Scripts para cálculos estadísticos rápidos y exploración descriptiva.
* **`visualization/`**: Configuraciones de temas minimalistas y funciones para gráficos recurrentes.

## 🚀 Funciones Destacadas

### 1. Reporte de Calidad de Datos
La función `get_data_report()` genera un resumen automático de nulos, duplicados y tipos de datos de cualquier DataFrame, facilitando el análisis exploratorio inicial (EDA).

### 2. Estandarización de Columnas
`clean_column_names()` elimina tildes, espacios y convierte a minúsculas todos los encabezados de un dataset, evitando errores comunes de sintaxis en Python.
